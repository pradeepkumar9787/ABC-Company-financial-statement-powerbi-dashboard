# 📊 ABC Company Financial Statement Dashboard | Power BI

## Project Overview

This project presents a professional Financial Statement Dashboard developed in Microsoft Power BI to analyze an organization's financial performance. The dashboard provides interactive insights into Revenue, Profitability, Assets, Liabilities, Cash Flow, and Financial Ratios.
